# gitconfig
My personal git configuration being used on my laptop as well as on servers I
have to deal with.

# Features
* colours everywhere
* many aliases to save your fingers!
* using vim for diff and merges
* git-sh-prompt setup

# Installation
```
	cp gitconfig ~/.gitconfig
```
